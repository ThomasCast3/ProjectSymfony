
# Symfony projet



ceci est le projet

## Installation

Install php with brew

```bash
brew install php  
```

Install symfony with brew

```bash
  brew install symfony-cli/tap/symfony-cli
```

Install composer

```bash
  composer install
```

Start symfony

```bash
  symfony server:start  
```
